# MEAN-Stack appBase from scratch

* Start MongoDb

* Install dependencies:

[Directory: _'working directory'_]

_cmd:_ 'npm install'

[Directory: _'/angular-src'_]

_cmd:_ 'npm install'

* Start server:

_cmd:_ 'npm start' / 'node server' / 'nodemon'

* Start Angular service:

[Directory: _'/angular-src'_]

_cmd:_ 'ng serve'

* Go to:
localhost:4200

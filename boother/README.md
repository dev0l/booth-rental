**Latest MEAN-Stack appBase from scratch**

* Start MongoDb

* Install dependencies:

[Directory: _'working directory'_]

cmd: 'npm install'

[Directory: _'/angular-src'_]

cmd: 'npm install'

* Start server:

cmd: 'npm start' / 'node server' / 'nodemon'

* Start Angular service:

[Directory: _'/angular-src'_]

cmd: 'ng serve'

Then go to:
localhost:4200

#Latest MEAN-Stack appBase from scratch

Make sure to have MongoDb running!

Using 'cmd' in Windows:

First install dependencies:

* Folder: '/boother v1.0'

'npm install'

* Folder: '/mean/angular-src'

'npm install'

Then start server:

'npm start'

*OR*

'node server'

*OR*

'nodemon'

Then start Angular service:

* Folder: '/mean/angular-src'

'ng serve'

Then go to:
localhost:4200
